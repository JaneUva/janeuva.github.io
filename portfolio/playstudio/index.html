About PlayStudio

PlayStudio brand uses bold color, shape, and typography to express its brand attributes: energy, education, and fun. The main objective was to create a branding system that would be appealing and delightful to the kids, so that parents fall in love with it too.

Web: PlayStudio website

The logo, illustrations and color system are all inspired by the elements that toddlers usually play with: from building blocks, crayons and musical instruments to different shades of paint colors.

Digital Solution

The site was developed around a user friendly CMS that allows PlayStudio to make updates when required.

Given that the website is aimed towards parents, it includes all the necessary information about the studio: working hours, classes, group sizes and contact. At the same time, it is maintaining colorful and playful brand identity.
