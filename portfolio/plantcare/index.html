Plantcare

Taking care of plants became so much easier with this incredible app. PlantCare helps you take care of your greenery and notifies exactly when & what to do. It has a large plant database and can give most efficient care tips.

Plant Identification

Users can identify plants quickly and easily by using camera identification feature. They simply need to take a snapshot of plant and PlantCare will do the rest. If for some reason, the plant was identified incorrectly, user can manually add the information and PlantCare database will be updated.

Design for this feature is based on the following principles:

  1. System should provide clear feedback during each search step, addressing errors in case of any and by displaing informative tooltips.
  2. Results communication should be clear and easy, e.g. in case of poor results, user should be prompted to search again or add custom option.
  3. Displaying search progress, awaiting results.



Reminders

Reminder is a simple and convenient way to always be aware of the condition of your plants and take care of them in time. The app will send daily reminders on things you need to tick off your gardening to do list.

When a notification arrives, an icon usually appears in the status bar. This signals to the user that there’s something to see in the notification drawer.



Plant Library & Collection

Users can scroll through a big collection of plants and learn more about each of it. They can also save plants as favourites and access them later in “My plants” page.



Search & Results

Sometimes users simply want to search for plant by name. Having search page allows to enter specific keywords and search results will be displayed. Each plant has a broad description and care tips, so it is easy for user to learn more about each plant.

Search text field with auto-complete suggestions is enables better user experience.
