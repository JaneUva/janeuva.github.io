Loopline Systems is a Berlin-based SaaS company with the mission of making performance review process as simple and effective as possible. Loop Now is an easy-to-implement cloud solution for agile management of continuous feedback, performance management and employee development. The tool focuses on objective management using OKRs, high-quality feedback, individual potential development and team engagement.

Data-driven design

Loop Now is designed to display a large amount of densely organized information in a easy-to-follow way. The UI focuses on presenting the data, it is clear and clean with an emphasis on typography, color, and shape.

Product Architecture

Loop Now has a hierarchical and sequential structure at the same time. Hierarchical - allows managers to look at their employee performance through a variety of categories. Sequential - is targeting a task-based structure, requiring employees to enter screens in a consecutive order. It is an ideal combination which allows all its users to explore content in an actionable way.

Loop Now navigation

On desktop and tablet, Loop Now uses rail navigation. It provides a permanent region to navigate between sections, while taking up little screen space. In a rail, each destination is represented by a unique icon. When a section is selected, a rail opens into a drawer and it‚Äôs icon becomes brighter and a section title is displayed.

On mobile, the navigation drawer is hidden and can be activated by tapping hamburger menu in the top left corner.

Focus on content

Content is didvided into different sections and takes the main space of the screen. It is done purposefully to involve user into content analysis and concentration. However, permanent position of rail, allows fast navigation.

Tabs on desktop

On desktop there is enough space for all tab items to be placed next to one another, horizontally.

Colors

Loop Now color theme has 2 primary colors in different intensities (dark, medium, light) and eight additional colors with different opacities. These are used to present reaable, distinct and vibrant infographics.

Color theme

The base primary background color is the same across all product. Two other primary colors appear in different shades and secondary colors are used for infographics and to represent other UI elements.
